# Amazon Clone

> An Amazon Clone application built with the MEVN stack, Algolia Search API, Stripe and AWS

## Install Dependencies

```
npm install OR yarn install
```

## Run App

```
# Run in dev mode
npm run dev OR yarn run dev

# Run in prod mode
npm start OR yarn start
```
